# Contenido del Laboratorio
El contenido del laboratorio varia entre problemas, desde matematicos, hasta de logica PHP.
- Problema #1: Cálculo de Área y Perímetro.
- Problema #2: Conversión de Pulgadas a Centímetros.
- Problema #3: phpInfo().
- Problema #4: Impresión de Autores.
- Problema #5: Sanitización, Normalización y Trimming de Datos.
- Problema #6: Calculadora básica.
